# Lancer le projet avec Docker Compose

## Prérequis

- Docker
- Docker Compose

## Structure du projet

.  
├── Backend/  
├── Frontend/  
└── docker-compose.yml

## Lancer le projet

1. Cloner le dépôt :

```bash
git clone https://github.com/IsAy15/DAD-Equipe-6
cd DAD-Equipe-6
```

2. Lancer le projet :

```bash
docker-compose up -d --build
```
